# AESTHERA

Elegant lifestyle transformation website featuring three phases: Rebuild, Refine, Radiate.

## Deployment

This project is designed for Vercel deployment.

1. Push to GitHub
2. Import to Vercel
3. Deploy automatically

## Local Development

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

## Important

- Place `ballerina.mp4` video in the `/public` folder
- Never modify the Whop checkout embed code
